# VA Claim Builder Version 4.2.0 RC6

RC6 is a defect-correction release candidate, not a final release.

- Native PDF extraction now precedes bounded local OCR.
- Raw, normalized, corrected, quality, method, and failure state persist per page.
- Paragraph wrapping and line-wrap hyphenation are reconstructed.
- OCR and Evidence bulk actions use stable checked IDs.
- Form 20-0995 Section 21A supports AcroForm and flattened text.
- Diagnosis/provider, timeline, nexus, and buddy imports create review drafts.
- Reanalysis preserves decisions and user corrections.

Project schema version is 11. Extracted results remain advisory and require
source verification.
