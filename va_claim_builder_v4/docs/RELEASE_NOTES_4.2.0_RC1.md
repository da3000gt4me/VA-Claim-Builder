# VA Claim Builder Version 4.2.0 RC1

RC1 hardens the complete Version 4.2 workflow through schema 10: project/document/OCR management, claims, evidence review and AI analysis, timeline, nexus, DBQ, rating strategy, optimizer, and submission packages. It adds validated local backup/restore, automatic pre-migration database snapshots, project validation and safe repair, coordinated job recovery/shutdown, rotating sanitized logs, diagnostic export, authoritative versioning, and reproducible PyInstaller configuration.

Exports include DOCX, CSV, JSON manifests, ZIP folders, and PDF merging where sources are readable. Existing schema 1–10 projects migrate in order; keep the automatic migration backup. This RC was built and tested only on the reported host. It is not a final production release and does not provide medical/legal advice or guarantee outcomes.
