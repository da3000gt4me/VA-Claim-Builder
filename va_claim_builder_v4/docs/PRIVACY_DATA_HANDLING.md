# Privacy and Data Handling

Projects, OCR artifacts, logs, temporary files, exports, and backups are stored locally at user-selected project paths or the platform application-data folder. Local-only mode prevents cloud provider invocation. Redact-before-cloud removes common identifiers before an enabled cloud request, but cannot guarantee anonymity. Logs and diagnostic exports exclude OCR text, statements, evidence content, provider payloads, API keys, and decrypted credentials. Package export is local and adds no telemetry.
